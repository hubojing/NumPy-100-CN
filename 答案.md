17. 下列表达式的结果是什么 (★☆☆)
```py
0 * np.nan
np.nan == np.nan
np.inf > np.nan
np.nan in set([np.nan])
0.3 == 3 * 0.1
```
结果：
```
nan
False
False
True
False
```
NaN具有不等于自身的属性，应该使用`np.isnan`来测试NaN值，`np.isnan(np.nan)`将产生True。
